Griffin Balducci's Project Portfolio

My Github Page
Book Rating Predictor

A recommendation system to predict user book ratings based on historical data. Leveraging machine learning models such as Random Forests, K-Nearest Neighbor, Neural Networks, Support Vector Machines, Naive Bayes, and Linear Regression to enhance prediction accuracy.

Crypto Day Trading Bot

Currently in progress, this Python-based crypto trading bot uses technical indicators like RSI, SMA, Bollinger Bands, and volume analysis to generate buy and sell signals. It pulls real-time market data from the Binance API and executes strategies to optimize trading opportunities. A paper trading feature is included to test strategies risk-free before going live, showcasing my skills in algorithm development, automated trading systems, and data analysis.

Recycling Reward Site

A website for tracking recycled plastics, allowing users to earn points redeemable for rewards and leaderboard rankings. Included features for tracking recycling efforts, an informational page about plastics, and promoting environmental awareness.

Simple Ciphers

This is a collaborative project involving the use of Caesar and Vigenere ciphers in the application of encryption and decryption of single words. Within the project, encryption/ decryption is visualized, encryption/decryption times are measured, and a bit of bruteforce/dictionary hacking is done to test the encryption strength of each cipher.

Gator AVL Tree

This is a project that was designed as an AVL (Self-Balancing Binary Search Tree) for organizing UF student accounts based on an ID number.

Image Processor

This program demonstrates image processing and modification of .tga image files. This program loads several input .tga files from the "input" folder, reads them, performs several pixel modifications, and outputs the new .tga images to the "output" folder.

Page Ranker

This project is a simplified version of Google's original PageRanking algorithm using an adjacency list.

DeltaMath2CanvasConverter

Currently in progress, this is a web-based application for automating gradebook updates by processing and merging Canvas CSV files and DeltaMath Excel files. Currently supports Python scripts in a virtual environment with plans to integrate a web interface.
